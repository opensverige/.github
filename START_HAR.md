# Start här — OpenSverige

**Vad vill du göra?** Välj en väg nedan. Alla community-länkar (Discord, IRL-sessioner): [opensverige.se](https://www.opensverige.se/).

**Första projektet:** Vi bygger en bra AI-agent för vår Discord — [discord-agent](https://github.com/opensverige/discord-agent). Eget repo; få invit till org, pusha fritt.

---

| Jag vill … | Gör så här |
|------------|------------|
| **Använda agenten / köra själv** | **Discord-agenten:** klona [discord-agent](https://github.com/opensverige/discord-agent), .env med nycklar, kör. Regler: [AGENT.md](AGENT.md), [agent-infra/](agent-infra/). |
| **Ändra en regel eller spec** | [Minimal väg](CONTRIBUTING.md#minimal-väg): öppna ett issue eller skriv i [Discord](https://www.opensverige.se/) med din idé; vi hjälper. [PR-väg](CONTRIBUTING.md#pr-väg): skapa branch → ändra → PR. |
| **Lägga till verktyg eller agent-roll** | Samma som ovan: issue eller [Discord](https://www.opensverige.se/), eller PR. Använd mallarna i `agent-infra/tools/_template.md` respektive `agent-infra/identity/_template.md`. |
| **Bara fråga eller diskutera** | [opensverige.se](https://www.opensverige.se/) — Discord och alla community-länkar. Vill du att det ska spåras → [öppna en issue](https://github.com/OPENSVERIGE_ORG/.github/issues). |
